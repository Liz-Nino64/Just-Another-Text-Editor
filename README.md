# J-A-T-E (Just Another Text Editor)

## Description

This application allows users access to a text editor, giving them the ability to create their own text files while both online and offline. Users can get as creative as they'd like, able to create amazing images of their own creations using simple characters.

## Installation

This application is available at https://j-a-t-e-8.herokuapp.com
